RPAL Online Interpreter

About

This is an online imlementation of RPAL Programming language. RPAL, the Right-reference Pedagogic Algorithmic Language, is a functional subset of PAL language.

Flags

-ast
Display Abstract Syntax Tree of the RPAL Expression.
-st
Display Standarized Tree of the RPAL Expression.
-noout
Don't display result of the RPAL Expression.
-l
Display Token list of the RPAL Expression.
